Hexo的简单使用

1、新建一个博客

新建完成后，指定文件夹的目录如下：

  • _config.yml：网站的配置信息
  • package.json：应用程序的信息
  • scaffolds：模版文件夹，当新建文章时，Hexo 会根据 scaffold 来建立文件
  • source：存放用户资源的文件夹
  • themes：主题文件夹，根据主题来生成静态页面

2、常用命令

  • init 新建一个博客
  • new 新建一篇文章

以上命令会创建一个 source/about/me.md 文件，同时 title 为 “About me”

  • generate 生成静态文件
  • server 启动服务器
  • deploy 部署网站
  • clean 清除缓存文件
